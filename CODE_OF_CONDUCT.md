# Code of Conduct

## AURA — Autonomous Universal Responsive Assistant

AURA is a system-level project built with a focus on **respect, clarity, and real contribution**.

All participants are expected to maintain a professional and constructive environment.

---

# 🤝 Our Standards

All contributors must:

* Be respectful in communication
* Be constructive in feedback
* Focus on improving the system
* Support a collaborative environment

---

# 🧠 Expected Behavior

* Communicate clearly and honestly
* Respect different ideas and approaches
* Give feedback based on system improvement, not personal opinion
* Help maintain a calm and focused development environment

---

# 🚫 Unacceptable Behavior

The following is not allowed:

* Harassment or personal attacks
* Toxic, aggressive, or disrespectful language
* Dismissing contributions without reason
* Misuse of the project or its purpose

---

# ⚙️ Project Culture

AURA is built with a **reality-first mindset**:

* No fake claims
* No misleading behavior
* No unnecessary ego-driven decisions

The goal is to build a **real system**, not to win arguments.

---

# 🛡️ Enforcement

Violations of this Code of Conduct may result in:

* warning
* restriction of contributions
* removal from the project

Decisions are made to protect the integrity of the system and its contributors.

---

# 🧭 Final Principle

Respect the people.
Respect the system.

Everything else follows.
